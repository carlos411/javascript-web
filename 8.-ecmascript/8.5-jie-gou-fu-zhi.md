# 8.5 解構賦值

## 解構賦值(Destructuring Assignment)

例 1：

```javascript
let a = 0;
let b = 1;

[a, b] = [b, a]; // 將原來 b 的值給變數 a；將原來 a 的值給變數 b
console.log(a);  // 1
console.log(b);  // 0
```



例 2：將陣列中，兩個項目對調：

```javascript
let arr = ["a", "b", "c", "d"];
[arr[1], arr[2]] = [arr[2], arr[1]];

console.log(arr); // ["a", "c", "b", "d"]
```
